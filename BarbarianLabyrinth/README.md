# Варвар в Лабиринте

Тёмно-фэнтезийная RPG на RPG Maker MV. Герой просыпается в чужом теле и спускается
в 50-этажный Лабиринт: расы и классы со своими навыками и древом развития, система
редкости и возвышения снаряжения, эссенции монстров, выживание (голод/усталость/рассудок),
мини-боссы каждые 5 этажей и боссы-Монархи каждые 10.

## Как играть
Скачайте сборку из раздела **Releases** (архив), распакуйте и запустите **Game.exe**.
RPG Maker устанавливать не нужно — всё необходимое уже внутри.

## Структура проекта (для разработки)
- `data/` — база данных игры (карты, враги, навыки, предметы, …), генерируется.
- `js/plugins/` — кастомные плагины + YEP.
- `img/`, `audio/`, `fonts/` — ресурсы (не хранятся в git; идут в сборке).

Данные собираются генераторами из папки `_build/` (Node.js): `node _build/rebuild.mjs`
пересобирает `data/*.json`, `node _build/gen_plugins.mjs` — список плагинов.

## Лицензия
Игровой контент — автора проекта. Стандартные ресурсы — по лицензии RPG Maker MV.
